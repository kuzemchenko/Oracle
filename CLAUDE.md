## Неизменяемые инварианты (нарушение = стоп работы)
1. **П8, нулевые выдумки:** каждый факт — со ссылкой, каждое число — с расчётом. «Нет данных» — обязательный и поощряемый ответ любого агента. Промпт каждого агента содержит это требование.
2. **П10, состязательность:** генератор, критик и судья — модели РАЗНЫХ семейств (config/models.yaml). Суд слепой, порядок аргументов рандомизирован, оценка по рубрике.
3. **П16, честность:** оценивать качество LLM-прогнозов на истории до cutoff моделей ЗАПРЕЩЕНО. Только форвард: каждый прогноз — по стандарту разрешимости (§9), запечатан (timestamp+hash) в journal/predictions.jsonl до исхода.
4. **Денежные ворота §11:** Сборка → Бумага (270 разрешённых прогнозов или письменная подпись пользователя) → Микроденьги ($500/идея). Kill-критерий не редактируется.
5. **Лимиты в коде:** config/limits.yaml — риск $3 000/мес, быстрые идеи $300/мес, бюджет $700/мес. Оркестратор проверяет их программно; обсуждать превышение в моменте — отказ.
6. **Математика — не LLM:** Brier, веса, Келли, FDR, индикаторы, лаги, запечатывание, сверка исходов — только детерминированный код в mathlib/ с тестами.

## Порядок сборки (план §24, gate каждого шага обязателен)
данные → mathlib → калибровка детерминированных компонентов на истории (§23.1) → промпты всех агентов → состязательный контур → дашборд → боевой запуск. Агентов раньше mathlib не писать.

## Архитектура исполнения
- Дирижёр (ты в интерактиве / orchestrator/run.py по cron) не имеет мнения о рынке: маршрутизация, поле суждений, карта противоречий, синтез, процедурное вето. На каждом синтезе пиши контрфактический протокол для абляции (§11.1).
- Воронка §6: скан с FDR → 30–50 кандидатов → фильтры → скоринг → дебаты топ-5–7 → выдача топ-3. «Стоящих идей нет» — легитимный результат.
- Каждая идея обязана ответить: «кто продаёт нам и почему он неправ», пройти тайминг (РАНО/ВОВРЕМЯ/ПОЗДНО/ЛОВУШКА) и антиманипуляционный балл.
- Формат отчёта — 13 обязательных полей §8, включая «что неизвестно» и рамку «исследовательский инструмент, не инвестиционная рекомендация».
