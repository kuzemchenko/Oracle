## 1. Детектор битых тиков

**Правило** (`mathlib/calibration/dataquality.detect_bad_ticks`, НЕ winsorize): день помечается битым тиком при ОДНОВРЕМЕННОМ выполнении трёх условий —
1. однодневный выброс: |log-доходность| ≥ 15%;
2. полный откат: цена следующего дня возвращается к уровню до выброса (|log(C[t+1]/C[t−1])| ≤ 6%);
3. идиосинкразия: в родственных инструментах в тот же день нет аномалии (max|доходность пиров| < 8%).

Без пиров инструмент не флагуется. Сплиты не ловятся (нет отката) — для них adjusted_close.

## 3. Фьючерс Brent на EODHD

**Недоступен на тарифе.** Проверка API:
- биржа фьючерсов `COMM` (`exchange-symbol-list/COMM`) → HTTP 404 (не в подписке);
- прямые тикеры `BRN.COMM`, `BZ.COMM`, `CO.COMM`, `B.COMM`, `BRENT.COMM`, `CL.COMM`, `BZ.NYMEX`, `BRN.ICE` → все 404;
- `search 'brent'` → только ETF (без фронт-месяца).

Единственный доступный Brent-инструмент — ETF-прокси **BNO.US**: последняя котировка close = 51.46 (2026-06-10), история 2876 баров с 2015-01-02 (≈11.4 лет). `brent_canonical_source` остаётся PENDING_USER (внешний источник цены Brent либо BNO-прокси с поправкой roll/contango).
