## H4 — АМПЛИТУДА (безусловная OLS-бета) · СТАТУС: ПОДТВЕРЖДЕНО

### (а) Механизм — да, безусловная OLS-бета по дневным лог-доходностям
`mathlib/cascade.py`:
- `ols_beta(x,y)` — `mathlib/cascade.py:58-84`: OLS y≈a+β·x, возвращает `r2=corr²` (:83).
- `node_sensitivity` — `:99-122`: β на синхронных рядах ЛОГ-ДОХОДНОСТЕЙ (`log_returns`, :35) закрытий,
  безусловная; `перенос_установлен` = Fisher-CI корреляции не включает 0.
- `node_amplitude(β,shock)=β·shock` — `:137-139`; `reliability_r2 = sens["r2"]` — `:216`.
- Боевой путь: `event_first_dryrun.py:77` → `cascade_from_quotes` → `node_cascade` (`:163`) → `node_sensitivity`.
  `node_cascade` (:171-181): sealable/probability/amplitude определяются **ТОЛЬКО безусловным**
  измерением; условная оценка (`conditional_sensitivity`, :125, этап Д3) — аддитивна, lazy,
  `with_conditional=False` по умолчанию, в бою НЕ включена (переключение — этап Э4(в,г), ещё не сделан).

→ Да: амплитуда/надёжность в бою = **безусловная** OLS-бета/R² по дневным закрытиям.

### (в) Отсев недостижимым верхним ярусом
Ярусы честности (`cascade.py:305` `_TIER_RANK`): A=эмпирический OLS (надёжный), B=структурный,
C=только механизм. Money-печать (sealable_path) требует ВСЕ звенья A; B/C → research-only (провизорный).
Из `граф_отбор.топ_k` по всем ef-прогонам (agent3_h4.py):
- кандидатов по низшему ярусу: **A=66, C=164** → **71.3% (164/230) кандидатов отсечены от money** в
  research-only, потому что несущее звено ярус C (только механизм, без ценового подтверждения).
- money-трек: 73 позиции — все ярус A; провизорный: 216 C + 8 A.

**Вывод H4:** амплитуда/надёжность в бою = безусловная OLS-бета·shock и R²=corr² по дневным лог-доходностям
(условная — не в бою). Типичный R²≈0.04 (median по всем каскадам). Верхний ярус A недостижим для 71%
кандидатов → они уходят в research-only. Плюс аномалия R²=1.0 у 14/24 money-печатей. **ПОДТВЕРЖДЕНО.**
